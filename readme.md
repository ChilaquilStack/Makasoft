# Makasoft Software Instructions

* Download project
  * `git clone https://github.com/Gizmo0oo/Makasoft.git`
  * `cd Makasoft`

* Install PHP dependencies
  * `composer install`

* Create project key
  * `php artisan key:generate`

* Create tables
  * `php artisan migrate --seed`

* Run server
  * `php artisan serve`

* Install node dependencies
  * `npm i production`

* Create app.js
  * `npm run prod`

* Run unit tests
  * `vendor/bin/phpunit`
